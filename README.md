# show-me-time

## install

### 1. node install 20.15.1
  - `curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash`
  - `source ~/.zshrc`
  - `nvm install 20`

### 2. git clone
  - `git clone https://github.com/honki12345/show-me-time.git show-me-time`
  - `cd show-me-time`
  - `npm install`
  - `node index.js`
